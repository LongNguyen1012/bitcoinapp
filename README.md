## Dependencies
- Node.js
- Firebase
- Stripe
- Paypal 
- Proof of Existence API
- Tesserect.js
- BioBERT
- hFigures Visualization
- Nodemailer

## Usage

#### Install the dependencies using npm (replace npm with 'yarn' if npm doesn't work)
npm install

#### And start the development server
npm start
